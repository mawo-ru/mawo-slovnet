## Описание оригинального проекта

SlovNet - это набор компактных и быстрых нейросетевых моделей для обработки русского языка. Проект является частью экосистемы Natasha и предоставляет модели для:

- **Named Entity Recognition (NER)**: Распознавание именованных сущностей (PER, LOC, ORG)
- **Морфологический анализ**: POS-теги, падежи, число, род
- **Синтаксический анализ**: Dependency parsing

### Основные особенности upstream-проекта:

- Компактные модели (2-3MB каждая)
- Высокая точность на новостных текстах
- Быстрая работа на CPU
- Обучение на новостных корпусах (Lenta, RIA, TASS)
- Интеграция с другими проектами Natasha

### Обучающие данные

Модели SlovNet были обучены на:

- **Lenta.ru** - российский новостной сайт
- **РИА Новости** - российское информационное агентство
- **ТАСС** - российское информационное агентство

## Улучшения MAWO

Команда MAWO добавила следующие улучшения к upstream-проекту:

- **Автоматическая загрузка моделей**: Модели скачиваются автоматически при первом использовании
- **Offline-first архитектура**: После загрузки работает полностью без интернета
- **Гибридный режим**: Нейросети + правила (fallback если модели недоступны)
- **Улучшенная интеграция**: Простой API для использования с другими библиотеками MAWO
- **Оптимизация памяти**: Эффективное использование ресурсов
- **Современный инструментарий**: Poetry для управления зависимостями
- **Расширенное тестирование**: Дополнительные тесты и примеры
- **Полная типизация**: Type hints для Python 3.10+
